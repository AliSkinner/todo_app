# Todo App
Classic Todo app build with React & Redux, saving data to Firebase & using GitHub OAuth.  

Production version on [Heroku](url:http://hidden-headland-90992.herokuapp.com/).

## Tech Used
* Front-end
 * React
 * Redux
 * ScSS
 * Foundation  


* Server
 * Express.js


* Database
 * Firebase


 * Testing
  * Mocha
  * Expect
  * Karma


* Tools
 * Webpack
 * Babel
 * Eslint
 * Yarn
